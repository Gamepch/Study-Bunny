# 클로버 키우기 게임 기획

## 개요
마이페이지에 내 클로버를 키우는 미니게임 추가. 공부 활동을 하면 클로버가 자라고, 방치하면 시든다.

---

## 클로버 성장 단계

| 단계 | 이름 | 조건 |
|------|------|------|
| 0 | 씨앗 🌱 | 시작 / 리셋 |
| 1 | 새싹 🌿 | 물 10회 이상 |
| 2 | 어린 클로버 ☘️ | 물 30회 이상 |
| 3 | 클로버 🍀 | 물 60회 이상 |
| 4 | 네잎 클로버 ✨🍀 | 물 100회 + 7일 연속 출석 |

---

## 물주기

마이페이지에서 버튼을 눌러야만 물을 줄 수 있음 (자동 지급 없음):

| 방법 | 물 횟수 |
|------|---------|
| 마이페이지 물주기 버튼 (하루 1회) | +2 |

---

## 방치 패널티

- 48시간 이상 아무 활동 없으면 클로버가 **시듦** (단계 -1)
- 시든 상태에서 추가 활동하면 회복 가능

---

## UI / 마이페이지 배치

```
[ 프로필 카드 ]
[ 연속 공부 & 잔디 ]
[ 🌱 내 클로버 카드 ]  ← 여기 추가
  - 클로버 이미지 (단계별 이모지 or SVG 애니메이션)
  - "물 주기 {N}회" 진행바
  - 다음 단계까지 N회 남음
  - 마지막 활동 시각 (방치 경고)
[ 내가 쓴 글 ]
```

## API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/clover` | 내 클로버 상태 조회 |
| POST | `/api/clover/water` | 수동 물주기 (하루 1회) |

물주기는 활동 시 서버에서 자동으로도 처리.

## 게임성 포인트

- 시들면 아쉬움 → 다시 공부하는 동기 유발
- 수동 물주기는 하루 1회 → 매일 방문 이유 생김
- 네잎 달성 시 카드 저장 → SNS 공유 바이럴

## 엔드게임: 네잎 클로버 카드

실물 네잎클로버 카드(길거리 파는 투명 라미네이팅 카드)에서 영감을 받은 디지털 수집 카드.

### 카드 디자인

```
┌──────────────────────────────┐
│  투명/반투명 카드 배경         │
│                              │
│        🍀                    │
│   (압화 느낌 클로버 일러스트)  │
│                              │
│  No. 003                     │
│  Clover Study                │
│                              │
│  이름    홍길동               │
│  기간    23일                 │
│  날짜    2026.06.07           │
└──────────────────────────────┘
```

- 배경: 반투명 유리(glassmorphism) 질감
- 클로버: SVG, 실물 압화처럼 약간 투명하고 잎맥 디테일
- 카드 번호: 내가 지금까지 키운 누적 n번째 클로버
- 기간: 씨앗 → 네잎까지 걸린 일수

### 카드 저장 흐름

1. 네잎 클로버 달성 → 축하 모달 등장
2. "카드로 저장하기" 버튼 → 카드 미리보기
3. PNG 다운로드 or 링크 공유
4. 새 씨앗으로 자동 리셋 → 다음 카드 도전

### 수집 갤러리

- 마이페이지에 "내 클로버 카드 n장" 섹션
- 카드들을 작게 그리드로 나열 (탭하면 크게 보기)
- 다른 유저 마이페이지에서도 카드 갤러리 공개 가능

### DB 추가

# clover_cards 테이블
